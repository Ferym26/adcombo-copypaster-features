Adcombo Copypaster features

Фичи упрощающие жизнь при копипасте прокл и лэндов

Что в коробке

Инструменты

gulp - сам Gulp.js
cssuno - удаление мертвого кода CSS
gulp-jsbeautifier - переформатирует html и css в нормальный общепринятый вид(пробелы, табуляция и т.д.)
gulp-autoprefixer - подстановка вендорских префиксов в CSS
cssnano - минификация CSS
babel - транспайлер ES2015

Для чего?

Автоматизировать нудные процессы

Практически всегда при копировании в файле css содержится довольно много мертвого кода.

Сжать проект

В конце работы над проклой или лэндом можно одной командой минифицировать css файл

Как начать пользоваться?

Скачать и установить необходимые инструменты

node.js - сам Gulp.js
git - git для windows(опционально, если не устраивает родная консоль)
